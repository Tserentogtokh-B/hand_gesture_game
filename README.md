# hand_gesture_game
Энэ тоглоом нь pygame болон mediapipe, opencv ашиглан хэрэгжүүлсэн гарын хөдөлгөөн ашиглан AI - ын эсрэг өрсөлдөж хонь хаших тоглоом юм. Та долоовор хуруугаа ашиглан өөрийн хоньчин хүүхдээ хөдөлгөж AI - аас өрсөж тодорхой тооны хонь хашаанд хийж чадвал ялалт байгуулна. AI нь manhattan distance - ийн дүрмээр өөртөө хамгийн ойр хонийг олж түүнрүүгээ явах зарчмаар ажиллана. Танд амжилт хүсье.
